## Como rodar o projeto

1. Usar **Node.js** na versão **20.x**.

2. Clone o repositório:

   ```bash
   git clone <URL_DO_REPOSITORIO>
   cd <NOME_DA_PASTA>
   ```

3. Instalação de dependências

   ```bash
    npm install
   ```

4. Iniciar o servidor

   ```bash
    npm run dev
   ```

5. Acessar o localhost no navegador

   ```bash
    http://localhost:3000

   ```
